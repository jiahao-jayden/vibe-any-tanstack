## 部署上线流程

<div className="fd-steps">
  <div className="fd-step">
    ### 推送代码
    将 `main` 分支推送到远程代码仓库（GitHub/GitLab/Bitbucket）。
    ```bash
    git remote set-url origin git@github.com:your-username/your-repo.git
    ```
    ```bash
    git add .
    git commit -m "feat: deploy to vercel"
    git push origin main
    ```
  </div>
  <div className="fd-step">
    ### 连接到 Vercel
    打开[Vercel](https://vercel.com/new)，点击`Import`，选择你的仓库完成导入。
    ![](/docs/deploy/vercel-import.webp)
  </div>
  <div className="fd-step">
    ### 配置环境变量（Environment Variables）
    导入你的`.env`文件，Vercel 会自动解析。
    ![Vercel Environment Variables](/docs/deploy/vercel-env.webp)
  </div>
  <div className="fd-step">
    ### 发起部署
    点击「Deploy」，等待构建完成。
  </div>
</div>
